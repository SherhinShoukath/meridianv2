# Meridian — AI-Native News Intelligence

> Depth-first news generated by an autonomous 6-stage AI editorial pipeline. Economist-level rigour. No human editors.

## Quick Start (Local)

```bash
npm install
ANTHROPIC_API_KEY=your_key npm start
# → http://localhost:3000
```

## Deploy to Railway (Free HTTPS — 2 min)

1. Push this folder to GitHub
2. railway.app → New Project → Deploy from GitHub repo
3. Add env var: `ANTHROPIC_API_KEY=sk-ant-...`
4. Done — Railway gives you a public HTTPS URL

## Deploy to Render (Free tier)

1. render.com → New Web Service → connect GitHub repo
2. Build: `npm install` · Start: `npm start`
3. Env var: `ANTHROPIC_API_KEY`

## Deploy to Fly.io

```bash
npm i -g flyctl && flyctl auth login
flyctl launch
flyctl secrets set ANTHROPIC_API_KEY=sk-ant-...
flyctl deploy
```

## Architecture
Browser → Express → /api/generate (SSE streaming, 6-stage pipeline)
                  → /api/qa (article Q&A)

## Pipeline
Topic Discovery → Source Synthesis → Fact Framework → Draft Generation → Editorial Pass → Publication
